# jaxlie

Matrix Lie groups for 2D/3D transformations in Jax.

Supports (exp, log, product, inverse, identity) operations for:

-[ ] Groups
    - [x] SO(2)
    - [x] SE(2)
    - [ ] SO(3)
    - [ ] SE(3)
- [ ] Tests
    - [x] Identity smoke test
    - [ ] log/exp
    - [ ] Inverse
    - [ ] Product
    - [ ] Apply
- [ ] CI
